# 🐟 AquaCalc — Калькулятор параметров воды

**Aquarium Water Calculator | Калькулятор реминерализации воды для аквариума**

Инструмент для точного расчёта дозировок солей при подготовке воды для аквариума. Помогает достичь целевых параметров GH, KH, TDS и ионного состава.

Tool for precise calculation of salt dosages for aquarium water preparation. Helps achieve target GH, KH, TDS, and ion composition parameters.

## 🧮 Входные параметры / Input Parameters

|Параметр|Описание|Default|
|-|-|-|
|GH|Общая жёсткость|6 °dGH|
|KH|Карбонатная жёсткость|2.0 °dKH|
|Объём|Количество воды|10 л|
|Ca:Mg|Соотношение кальция к магнию|4:1|

## 🧪 Источники солей / Salt Sources

### Кальций / Calcium

* CaSO₄·2H₂O (гипс)
* CaCl₂ (безводный)
* CaCl₂·2H₂O (дигидрат)
* CaCl₂·6H₂O

### Магний / Magnesium

* MgSO₄·7H₂O (эпсом)
* MgSO₄ (безводный)
* MgCl₂·6H₂O

### Калий / Potassium

* K₂SO₄
* KCl
* K₂CO₃
